# DeerTheGreat-Network-Scanner-V.04
DeerTheGreat-Network-Scanner DeerTheGreat is a fast, futuristic network port scanner built from scratch with zero dependencies. It does TCP connect scanning like nmap -sT, with a live animated terminal UI. Scans single hosts, IP ranges, or CIDR subnets, identifies services, and grabs banners. Pure Python, no pip installs, works on Linux and Windows
